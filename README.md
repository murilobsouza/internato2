
# Sistema de Check-in de Presença

Este projeto está pronto para deploy automático.

## Como Publicar:
1. Crie um repositório no GitHub.
2. Envie estes arquivos para lá.
3. No site da [Vercel](https://vercel.com), importe o repositório.
4. O link será gerado automaticamente.

## Limitação Atual:
O sistema utiliza `localStorage`. Para que o professor veja os dados de múltiplos alunos, é necessário integrar um serviço de banco de dados (Firebase, Supabase ou uma API).
